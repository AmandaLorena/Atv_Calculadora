Projeto - Calculadora

Requisitos

O objetivo do presente projeto é desenvolver uma calculadora, cujas operações possíveis são soma (+), multiplicação (*), subtração (-), divisão (/) e raiz quadrada (√). O usuário deseja um layout com critérios de usabilidade sendo atendidos, para realizar atividades com segurança no dia-a-dia, através de react native e expo.

RF001: Deve permitir realizar operações matemáticas com até 2 argumentos, retornando 1 informação.

RF002: Deve conter campo de entrada de dados para inserção de valores.

RF003: Deve realizar as operações através de algoritmos alfanuméricos.

RF004: Deve exibir os resultados na tela.

RF005: Deve possuir botões para os números e operações matemáticas.

RF006: Deve permitir limpar os valores inseridos na tela.
